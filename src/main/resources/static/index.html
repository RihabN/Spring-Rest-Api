AMS API 1.0

Article Management System REST API

Available Endpoints:

GET /providers/list - Get all providers
POST /providers/add - Create provider
GET /providers/{id} - Get provider by ID
PUT /providers/{id} - Update provider
DELETE /providers/{id} - Delete provider

GET /articles/list - Get all articles
POST /articles/add/{providerId} - Create article
PUT /articles/update/{providerId}/{articleId} - Update article
DELETE /articles/delete/{articleId} - Delete article
